Sobre o Software

Desenvolvido por Leandro Souza, o Sistema de Controle de Patrimônio é uma solução desktop completa para instituições que buscam organizar o inventário de hardware e mobiliário com precisão.

O software automatiza desde o registro de entrada até a baixa de ativos, gerando etiquetas inteligentes com QR Code para consulta rápida e relatórios profissionais em PDF.

Stack Tecnológica

  • Python 3.13 - Núcleo de alta performance
  • CustomTkinter - Interface Moderna (Dark Mode)
  • SQLite3 - Banco de dados local seguro
  • QR Code Engine - Rastreabilidade em tempo real
  • ReportLab - Automação de documentos e etiquetas

Diferenciais do Sistema

Segurança Multinível

Acesso restrito via login, recuperação de senha por e-mail e chave mestre de hardware.

Etiquetas Inteligentes

Geração automática de etiquetas em lote com QR Code integrado para auditorias rápidas.

Update Silencioso

Integração com GitHub para verificação automática de novas versões disponíveis.
