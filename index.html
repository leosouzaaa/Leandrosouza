Gestão Inteligente

Desenvolvido por Leandro Souza, este sistema foi projetado para eliminar o caos no inventário de instituições públicas e privadas.

Com uma interface 100% intuitiva em Dark Mode, o operador consegue registrar entradas, gerar etiquetas em lote e realizar baixas em segundos, tudo com backup local automático.

Tecnologias

Python 3.13 SQLite3 CustomTkinter ReportLab PDF

QR Code Automático

Geração de etiquetas térmicas ou A4 com códigos rastreáveis para auditoria instantânea por smartphone.

Recuperação por E-mail

Integração nativa com Gmail SMTP para envio de credenciais e recuperação segura de conta admin.

Updates via GitHub

Sistema inteligente que verifica novas versões no repositório oficial, mantendo o software sempre atualizado.
