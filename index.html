Como funciona o pagamento único?
Você paga apenas uma vez R$ 149,90 e o sistema é seu para sempre. Não existem mensalidades, anuidades ou taxas de manutenção ocultas.
Preciso de internet para usar o sistema?
Não. O sistema foi desenvolvido para funcionar totalmente em modo offline. O banco de dados fica salvo no seu computador, garantindo velocidade e privacidade total dos seus dados.
O sistema gera etiquetas com QR Code?
Sim! O sistema gera automaticamente etiquetas no formato 40x20mm com QR Code exclusivo. Você pode imprimir individualmente ou em lote para identificar todo o seu patrimônio.
Qual a capacidade de itens do sistema?
O motor de banco de dados SQLite é extremamente robusto. O sistema suporta tranquilamente mais de 50.000 itens cadastrados sem perder a velocidade de busca.
Consigo exportar relatórios em PDF?
Sim. Em todas as telas você tem o botão para gerar listagens em PDF com um clique, facilitando conferências, auditorias e balanços patrimoniais.
